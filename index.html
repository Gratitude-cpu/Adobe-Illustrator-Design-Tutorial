Welcome to T-shirt Design Tutorial

This tutorial is at the intermediate level focusing on mainly silhouette illustration and the use of shapes. Throughout this design, there are many elements that beautifully showcase California!

Step 1

This is the initial setup for the T-shirt design. Adobe illustrator is the software that will be used to create this T-shirt design.

Step 2

Sketch out this design on a piece of paper. You may trace the outlines of this picture too. After sketching, upload the sketch to Adobe Illustrator.

Step 3

Now, using the curvature tool boxed in red, trace out the rocky mountains. Make sure to set the fill to transparent while tracing. Set the stroke to a black color and a stroke weight of 3 pt.

Step 4

The next step is to fill in the rocky mountains in black. To do this, first click on the rocky mountain and then the box that the arrow is pointing to. Then, using the eyedropper tool, click on the black stroke you just created. A rocky mountain should get filled. Repeat this step for the other mountain as well.

Step 5

Similar to step 2, outline the poppies and the meditative woman figure in black using the curvature tool. Set the stroke weight to 3 pt. Remember to make sure that the fill is transparent while outlining like this picture.

Step 6

Similar to how the rocky mountains are filled, the poppies, the meditative woman figure, and the heart will be filled in. Start of with each poppy, by selecting each one, then clicking on the box the arrow is pointing to. After that, use the eyedropper tool to click on the black rocky mountains, and the object you have initially selected would be filled. You can use this same method to color in the meditative woman. As you can see, a few of the poppies are golden brown color. To color in these poppies, select the poppy, then the box the arrow is pointing to, and this time double tap. An hexcode bar would open, here, replace the code with this "#7E491E". This hex code will turn the poppy to golden brown color. Similarly, select the outlined heart, and double tap that box, and replace the hex code with this "D24427". This turns the heart in the red color. Remove the stroke of the heart by clicking on the box next to the one that you just double tapped.

Step 7

Using the curvature tool, outline the grizzly bear. Then select the grizzly, then double click on the box the red arrow is pointing. A hex chart will open up, and paste in this hex code: "7E491E".

Step 8

Next step is to color in the grizzly bear. To do this, select the grizzly bear, double tap the box that the arrow is pointing to, and when the hexcode chart opens up, paste this hexcode in: "7E491E". You can also use the eyedropper tool method to accomplish this task as well from the previous steps.

Step 9

Using the eye dropper tool, select the stroke color to be black and 3.5 pt weight. Then use the blob brush tool to paint over the sketch strokes of both of the rocky mountains. Similar to how the other elements are done, outline and fill in the trouts the same color as the grizzly bear. Keep in mind that one of the trouts is black.

Step 10

Now, open up the window that you have setup in the first step. Using the shape tool, boxed in red, draw a big yellow box covering up the whole artboard. The hexcode for that is: #FFE07F. Adjust all the elements as needed according to this picture.

Step 11

Paste all the elements from the other window onto this light yellow background.

Step 12

Use the shape tools boxed in red and yellow to create the flower-like shape above the grizzly bear. The shape tool boxed in red can be used to create circle. And the shape tool boxed in yellow can be used to create the triangles. The hex code color for the flower-like thing is: "D24427". To get the text, go to the tabs on the navigation bar at the top of Adobe Illustrator, you will find a tab called "Type". Then on the drop-downs, you will find "Fonts". Then select "Bahnschrift". Type in the title in the font. Choose an appropriate size or set a stroke to it. You can choose the same hexcode as the flower like thing for the words in red. For the words in blue, here is the hexcode: "213D78". Using the same shape tool boxed in red, you can draw a black cricular thing. And create blue rectangles on top of it. This part does not have to be perfect as the tech companies in the silicon valley are all over the place. The blue boxes represent the tech companies. The blue color is the same as the blue text.

Step 13

Be sure to add the Silicon Valley text in the same font. Congratulations! The t-shirt design is finished!!!
